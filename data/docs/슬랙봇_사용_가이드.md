# eNomix 공수 관리 슬랙 봇 사용 가이드

## 📌 개요
슬랙에서 `@enomix-effort` 봇을 멘션하여 과거 프로젝트 공수 정보를 조회할 수 있습니다.

---

## 💬 기본 사용법

### 봇 호출 방법
슬랙 채널이나 DM에서 `@enomix-effort` 멘션 후 질문을 입력합니다.

```
@enomix-effort ENOMIX-1234 프로젝트 공수는?
```

---

## 📊 질문 유형

### 1. **프로젝트 전체 공수 조회** (Epic 단위)

프로젝트 전체 공수를 확인하려면 **반드시 다음 키워드 중 하나를 포함**해야 합니다:
- `프로젝트`
- `epic` 또는 `에픽`
- `전체 공수`
- `프로젝트 공수`

#### 올바른 예시 ✅
```
@enomix-effort 경남은행 프로젝트 공수는?
@enomix-effort ENOMIX-1234 epic 정보
@enomix-effort 삼성카드 전체 공수 알려줘
@enomix-effort 케이뱅크 프로젝트 공수
```

#### 잘못된 예시 ❌
```
@enomix-effort 경남은행 공수는?        (키워드 없음 → 개별 작업 검색됨)
@enomix-effort ENOMIX-1234 알려줘     (키워드 없음 → 개별 작업 검색됨)
```

### 제공되는 정보
- Epic 제목 및 Jira 링크
- 총 공수 및 작업 개수
- 단계별 공수 (세팅, 분석/설계, 구현, 테스트, 반영/이행, 기타)
- 담당자별 공수 (2명 이상 투입 시)
- AI가 생성한 프로젝트 요약
- 고객사 특성 분석 (있는 경우)
  - 난이도 등급 및 범위
  - 요구사항 명확성, 개발 유연성, 고객 소통 지수
  - 버퍼 필요 여부 또는 표준 공수 충분 여부
  - 참고자료 링크
- 주요 작업 목록 (상위 10개)

---

### 2. **개별 작업 조회** (Task 단위)

특정 작업이나 키워드로 관련 작업을 검색합니다.

#### 예시
```
@enomix-effort 도메인 추가 작업
@enomix-effort CS톡 관련 개발
@enomix-effort 카카오상담톡 설정
```

### 제공되는 정보
- 유사한 작업 목록 (최대 5개)
- 각 작업의 Jira 티켓, 제목, 공수, 담당자

---

## 🎯 도움말 명령

언제든지 사용법이 궁금하면 다음 명령어를 입력하세요:

```
@enomix-effort help
@enomix-effort 헬프
@enomix-effort 가이드
```

간단한 사용법 안내 메시지가 표시됩니다.

## 📝 유용한 팁

### 1. 프로젝트 전체 공수를 보려면
**"프로젝트", "epic", "전체 공수"** 중 하나를 꼭 넣으세요!

```
✅ "경남은행 프로젝트 공수"
❌ "경남은행 공수"
```

### 2. Epic 키로 직접 검색
Epic 키(ENOMIX-1234)를 알고 있다면 정확하게 입력하세요.

```
@enomix-effort ENOMIX-1234 프로젝트 정보
```

### 3. 고객사명으로 검색
고객사명만 입력해도 관련 프로젝트를 찾을 수 있습니다.

```
@enomix-effort 삼성카드 프로젝트
```

### 4. 피드백 남기기
봇의 답변에 이모지로 피드백을 남길 수 있습니다:
- 👍 (`:thumbsup:`): 도움이 되었어요
- 👎 (`:thumbsdown:`): 도움이 안 되었어요

피드백은 시스템 개선에 활용됩니다.

## ⚠️ 주의사항

### 1. Epic 키워드 필수
프로젝트 전체 정보를 보려면 반드시 `프로젝트`, `epic`, `전체 공수` 같은 키워드가 필요합니다.

### 2. 봇 멘션 필수
`@enomix-effort`를 먼저 입력해야 봇이 응답합니다.

### 3. 응답 시간
- 개별 작업 조회: 2~5초
- 프로젝트 전체 조회: 5~10초

복잡한 질문은 시간이 더 걸릴 수 있습니다.

### 4. 데이터 범위
시스템에 수집된 완료된 Epic 데이터만 조회 가능합니다. 진행 중이거나 수집되지 않은 데이터는 조회되지 않습니다.

---

## 🐛 문제 해결

### 봇이 응답하지 않음
- `@enomix-effort`를 정확히 멘션했는지 확인
- 슬랙 워크스페이스에 봇이 추가되어 있는지 확인
- 채널에 봇을 초대했는지 확인 (`/invite @enomix-effort`)

### 원하는 정보가 나오지 않음
- 프로젝트 전체 정보를 원한다면 "프로젝트", "epic" 같은 키워드 추가
- Epic 키나 고객사명을 명확히 입력
- 질문을 더 구체적으로 작성

### "관련 정보를 찾을 수 없습니다" 메시지
- 해당 Epic이 시스템에 수집되지 않았을 수 있습니다
- 웹 페이지(http://211.63.24.116:9000)에서 "Epic 하위 작업 일괄 가져오기"로 수동 추가 필요
